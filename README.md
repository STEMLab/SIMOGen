## Getting Started
The follwing tools are required to build OpenGDS/Conversion:
* Java Development Kit 8 (JDK 8)
* [Apache Maven 3.x](https://maven.apache.org/download.cgi)

```
1. clone this project
2. mvn install
3. mvn exec:java
```
